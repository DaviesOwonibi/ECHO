# E.C.H.O or Enhanced Cognitive Helper and Organizer
E.C.H.O is a personal project I made to help me manage my life on my computer, its also made to illustrate how far we as humans have come, where we can use our devices without having to touch our mouse or keyboard at all as it is voice activated. I made it in python, using the SpeechsRecognition and Python Text To Speech(PyTTSX3) libraries.
